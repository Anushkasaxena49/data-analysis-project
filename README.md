# 📊 Data Analysis Project (Student Marks Analysis)

## 🧾 Project Description
This project analyzes students' marks using Python and Pandas. It helps in understanding performance of students by calculating key statistics from the dataset.

---

## ✨ Features
- 🧮 Calculates average marks  
- 📈 Finds highest and lowest marks  
- 🏆 Displays top performing students  
- 📊 Simple data analysis using Pandas  

---

## 🛠️ Tech Stack
- 🐍 Python  
- 📦 Pandas  

---

## 🚀 How to Run
- Open the project in VS Code or Jupyter Notebook
